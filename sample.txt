Step 3: Create a New Lambda Function

Step 4: Create a DynamoDB Table

Before writing the code, go to DynamoDB → Create table

Partition key → unique (String)

Step 5: Upload / Write the Lambda Code
Replace the default code with this:
pythonimport boto3
from uuid import uuid4

def lambda_handler(event, context):
    dynamodb = boto3.resource('dynamodb')
    dynamoTable = dynamodb.Table('newtable2')

    if 'Records' in event:
        for record in event['Records']:
            bucket_name = record['s3']['bucket']['name']
            object_key  = record['s3']['object']['key']
            size        = record['s3']['object'].get('size', -1)
            event_name  = record['eventName']
            event_time  = record['eventTime']

            dynamoTable.put_item(
                Item={
                    'unique':    str(uuid4()),
                    'Bucket':    bucket_name,
                    'Object':    object_key,
                    'Size':      size,
                    'Event':     event_name,
                    'EventTime': event_time
                }
            )
    else:
        print("No S3 event received")

    return {
        "statusCode": 200,
        "body": "Data stored in DynamoDB"
    }
What this code does: Each time S3 triggers this function, it loops through the event records, extracts the bucket name, file name, size, event type & timestamp, and stores everything as a new row in DynamoDB with a unique ID.

Event type → All object create events
Click Add

Step 8: Test Your Function

Run it and check if DynamoDB gets a new record
